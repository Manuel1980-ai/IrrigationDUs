IrrigationDUs is an application designed to optimize agricultural irrigation management. It allows users to monitor and ensure the proper functioning of irrigation systems across various crops, such as olive trees, vineyards, and apple trees.

The tool facilitates the quick identification of dripper uniformity, contrasting it with the original design to verify compliance with seasonal irrigation objectives. Adaptable to both the imperial and metric systems, the interface is available in Spanish, English, French, German, and Chinese.

Key Features:

Georeferenced Monitoring: Mark a PIN at each monitoring station to generate a detailed historical record of the exact location and time of measurements.

Data Export: Save measurements from every point and easily download them in Excel format for subsequent professional analysis.
